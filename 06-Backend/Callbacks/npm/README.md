Calculadora UDLA
Este paquete nos ayuda a sumar restar y multiplicar
```javascript
exports.CalculadoraUdla={
    sumar: (n1,n2)=>{
        return n1+n2;
    },
    restar:(n1,n2)=>{
        return n1-n2;
    },
    multiplicar:(n1,n2)=>{
        return n1*n2;
    }
};

```
